# 數學拔河遊戲網站

## Context
設計一個雙人對戰的數學學習網站，兩位學生各操作一個數字鍵盤，透過快速答題搶分，中間以拔河動畫呈現雙方分數差距，增加競爭趣味。

---

## 技術選型
- 純 HTML + CSS + JavaScript（單一 `index.html` 檔案，無需任何框架或後端）

---

## 版面配置

```
[ 左側玩家區 ]  [ 中間拔河動畫 ]  [ 右側玩家區 ]
  題目: 5+3=?      🟥━━🙂━━🟦       題目: 7×4=?
  [數字鍵盤]     分數差距動態繩子    [數字鍵盤]
  目前輸入: _                        目前輸入: _
```

每側玩家區包含：
- 題目顯示區（大字體）
- 計時進度條（視覺化顯示剩餘分數）
- 數字鍵盤（0–9、退格、送出）
- 目前輸入顯示
- 答錯時的錯誤訊息
- 累積總分

---

## 題目生成規則
- 加法：兩個 1–20 的數字
- 減法：兩個數字，確保結果為正整數
- 乘法：兩個 1–9 的數字
- 除法：先選商（1–9）和除數（2–9），再算被除數，確保整除

---

## 計分邏輯
- 題目出現時開始計時
- 分數 = max(1, 5 - floor(經過秒數 / 5))
- 答錯：顯示「答錯了！再試一次」，不計時重置，繼續扣分
- 答對：加分，立即出下一題

---

## 拔河動畫
- 中間顯示一條繩子（CSS 繪製）
- 繩子中間有一個旗幟或標記，根據左右總分差距左右移動
- 分數相同時旗幟居中；左方領先則旗幟往右偏（對右方施壓）
- 動畫平滑過渡（CSS transition）
- 兩側各有一個卡通人物圖示（用 emoji 或 CSS 繪製）

---

## 遊戲流程
1. 頁面載入 → 顯示「開始遊戲」按鈕
2. 按下開始 → 雙方同時出現第一道題（各自獨立、不同題目）
3. 兩側獨立計時、獨立答題
4. **勝利條件**：任一方總分領先對手 **10 分**時，遊戲結束，顯示獲勝畫面
5. 可按「再玩一次」重置

---

## 檔案結構
```
index.html   ← 所有 HTML / CSS / JS 全部在此單一檔案
```

---

## 驗證方式
1. 用瀏覽器開啟 `index.html`
2. 確認：
   - 兩側題目獨立出現，各自計時
   - 答錯時顯示錯誤訊息，不切換題目
   - 計時超過 5/10/15/20 秒時分數正確為 4/3/2/1
   - 答對後分數正確累積，拔河繩子動畫位移
   - 行動裝置橫向也能正常顯示（響應式）
